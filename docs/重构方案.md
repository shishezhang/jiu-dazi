# 🍺 酒搭子 v2.0 — 纯静态展示方案

> 目标：将现有 UniApp 项目重构为零依赖的纯静态展示站，一页展示产品全貌，可直接部署到任何静态托管（Vercel/Netlify/GitHub Pages）。

---

## 一、为什么重构

| 旧方案（UniApp） | 新方案（纯静态） |
|------------------|------------------|
| 依赖 UniApp 框架 + 300MB node_modules | 零依赖，一个 HTML 文件跑起来 |
| 需要编译构建 | 浏览器直接打开 |
| 面向 App/小程序 | 面向 Web 展示 |
| 代码改动需要重编译 | 改完刷新即见 |
| 部署需要 build 产物 | 拖拽上传就能部署 |

---

## 二、技术方案

```
纯 HTML5 + CSS3 + Vanilla JS（零框架零依赖）

├── HTML      语义化结构，SEO 友好
├── CSS       暗黑鸡尾酒主题，CSS 变量驱动
│             Grid/Flexbox 布局，全响应式
└── JS        轻量交互：轮播/动画/表单校验
              Mock 数据驱动展示
```

## 七、交互细节

| 功能 | 实现方式 |
|------|----------|
| 导航高亮 | IntersectionObserver 检测当前 Section |
| 截图轮播 | CSS scroll-snap + JS 自动播放 |
| 数字递增 | 进入视口时从 0 滚到目标数 |
| 视差滚动 | CSS transform: translateY() + scroll |
| 卡片悬停 | CSS transition + box-shadow glow |
| 毛玻璃效果 | backdrop-filter: blur() |
| 下载弹窗 | 点击 CTA 弹出模态窗 |
| 平滑滚动 | scroll-behavior: smooth |

## 八、响应式策略

```
> 1024px   桌面端    全宽展示，4 列卡片
 768-1024  平板端    2 列卡片，手机预览居中
< 768px    手机端    单列，隐藏手机外框，截图直接展示
```
